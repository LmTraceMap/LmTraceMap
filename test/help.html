Step 1 : uploading

there have two methods to upload your file
1. click "select file" button, as shown in the figure above.
2. drag your file into window, as shown in the figure below.

progress bar will display your uploading progress.

fill the form.
"L.monocytogene's country" please select the country which the gene be found, "your email" please fill the email to receive the result page.

When the form is be filled, and the upload is complete, click "submit" button to submit.

Step 2 : wainting for calculation

user can close the window,
if calculation is complete, system will send an email to you.
inside the mail, there have a link, directing to result page.

Step 3 : view result page

system will compare the gene uploaded by user to others, then display the top 50 matched gene's info.

    ● map page

  • display the top 50 matched gene's location.

  • the red location is the country which user uploaded gene be found.
    clicking yellow circle, you can see the place name, gene name, and the result of the comparison.

    ● table page

  • display the top 50 matched gene's info as table.

  • clicking "download .csv" or "download .xls" button, you can download table to your device.

  • click "choose column" button, you can choose the column which you wanted.
    if you want to display "strain", then click "strain" in drop-down menu ;if you want to hide "strain", then click "strain" again to hide it.
    p.s. "rank","gene name","distance" is basic column, cannot be hidden.

  • clicking column name, the data will be sorted.
